Run the application with this command: ./run-cookie-utils.sh -f $fileName -d $date
for example: ./run-cookie-utils.sh -f cookie_log.csv -d 2018-12-09

Technologies used:
Language: Java 11
Build: Maven
Library: logback & junit dependencies

To build the application from the source:
1. Extract cookie-utils-src.zip
2. cd cookie-utils-src
3. mvn clean install
4. The application archive can be found at target/cookie-utils.zip
